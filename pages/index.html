Beatify

Beatify je web stranica za streaming muzike. Ime Beatify dolazi iz inspiracije od Spotify-a.

Web stranica je statična (napravljena samo sa HTML-om i CSS-om) i ne sadrži Javascript.

Web stranica sadrži minimalan izgled i broj stranica.

Ovu stranicu je napravio Patrik Raspudić (patrikraspudic@gmail.com) u svrhe HTML/CSS projeka za sveucilisni studij Razvoj Web Aplikacija.
